# Online Banking Application (Spring Boot)

A mini online banking backend application built using Spring Boot and Hibernate.
It supports account management and transaction workflows with validations.

## Features
- Create bank account
- View account details
- Deposit money
- Withdraw money
- Transfer money between accounts
- Transaction history
- Input validations and error handling

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA (Hibernate)
- H2 In-Memory Database
- Maven

## How to Run
1. Clone the repository
2. Open in IDE (IntelliJ / Eclipse)
3. Run `BankingApplication.java`
4. Application runs on `http://localhost:8080`
